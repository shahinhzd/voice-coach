Link a shared folder

Paste the sharing link of a Google Drive folder. Every transcript you save will be downloaded and you can drop it straight into that folder — or if you have Google Drive desktop sync, it appears automatically.

How to get the folder link:
1. Open Google Drive
2. Right-click the folder you want → Share → Copy link
3. Set access to "Anyone with the link" (so the app can open it)
4. Paste the link above and click Connect
